# 🎧 Headphones Webpage HTML & CSS Project

A fully responsive, accessible webpage built from scratch using only HTML and CSS, based on a Figma design provided by ALX.



## Project Overview

This project is part of the ALX Front-End Foundations curriculum.  
The objective is to recreate a professional webpage identical to the Figma design without using any external CSS frameworks or JavaScript.

The skils will be demonstrated in:

- Semantic HTML  
- CSS layout positioning  
- Flexbox & responsive design  
- Accessibility  
- Implementing UI from Figma



##  Figma Design

The web page must visually match the provided mockup.

Designed by Nicolas Philippot  


##  Requirements

### Technical Rules
- No JavaScript allowed  
- No external CSS frameworks (no Bootstrap, Tailwind, etc.)  
- Only custom HTML & CSS  
- Use the required fonts: Source Sans Pro and Spin Cycle OT  
- Maximum content width: 1000px (centered)  
- Must match the Figma design as closely as possible (rounding small float values allowed)

###  Interactions
Links hover	:#FF6565
Buttons hover:opacity 0.9
Responsive Layout	Screen ≤ 480px → mobile design


## 📱 Responsive Design Behavior

The webpage must adapt smoothly:

- Desktop version*→ Default layout  
- Mobile version (≤ 480px) → Stacked layout, centered elements, mobile spacing  

Responsiveness is implemented using:

- Media queries  
- Flexible containers  
- Percentage-based widths


## Features Implemented

-  Pixel-perfect recreation of the Figma design  
-  Custom-designed interface using pure HTML & CSS  
-  Fully responsive (desktop → mobile)  
-  Accessible markup using ALT text, ARIA labels, proper headings  
-  Navigation with hover states  
-  Buttons with active and hover effects  
-  Clean project structure
